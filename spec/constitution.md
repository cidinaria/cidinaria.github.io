# Constitution
## Goal
Create a personal academic and professional website in English for Maria Cidinaria Silva Alves.
## Technical rules
- Use only HTML, CSS and JavaScript vanilla
- Build a static single-page website
- Make it compatible with GitHub Pages
- No backend
- No database
- No authentication
- No API keys in the code
- Keep the site lightweight, responsive and accessible
## Quality rules
- Professional and science-focused visual identity
- Mobile-first responsive layout
- Clean semantic HTML
- Basic SEO optimization
- Easy maintenance and future editing
- Fast loading and simple structure
## Content rules
- The website must be written in English
- The content must reflect an academic and scientific profile
- The tone must be credible, clear, and professional
- The website must highlight research, publications, and technical expertise
## Security rules
- Do not expose secrets or credentials
- Do not include unnecessary third-party scripts
- Do not collect sensitive information
- Keep contact options simple and direct
