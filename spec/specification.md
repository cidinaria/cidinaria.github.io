# Specification
## Objective
Create a personal website to present Maria Cidinaria Silva Alves as a biologist and bioinformatician.
## Audience
- Researchers
- Academic collaborators
- Institutions
- Recruiters
- Scientific partners
## Website type
A single-page professional and academic portfolio website.
## Main sections
- Hero
- About
- Expertise
- Experience
- Education
- Skills
- Selected Publications
- Contact
## Main goals
- Present academic and professional profile clearly
- Highlight expertise in bioinformatics, genomics, and omics
- Showcase selected publications
- Offer clear contact options
- Link to GitHub and Lattes
## Functional requirements
- Responsive navigation menu
- Smooth scrolling between sections
- Clickable contact links
- Clean section hierarchy
- GitHub Pages ready structure
## Non-functional requirements
- Fast and lightweight
- Easy to edit manually
- Readable on desktop and mobile
- Professional design suitable for science and biotechnology
## Out of scope
- Blog system
- Admin panel
- Private login area
- Backend forms
- Payment features
